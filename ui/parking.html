General features

DD Talk — онлайн сервис защищенной видеовстречи между двумя или несколькими анонимными пользователями.

В сервисе нет баз данных и не собирается персональная информация. Участники включаются в защищенную пиринговую сеть и подключаются друг к другу анонимно без какой либо регистрации с помощью уникальной ссылки, которую они открывают в браузере. Это безопасно поскольку данные между участниками передаются в шифрованном виде (mesh network). Сервис - универсальная альтернатива огромному числу коммуникаторов: Viber, WhatsApp, Skype, Telegram, Duo, Facebook Messenger, Google Hangout и т.п. DD Talk помогает удобно и быстро организовать групповую приватную онлайн встречу, в ситуации когда людям сложно договориться о выборе одного средства связи.

С DD Talk групповые встречи сделать гораздо проще. Для этого один из участников открывает в DD Talk комнату и рассылает ссылку для подключения (или сообщает только ее название) всем остальным. Для рассылки инициатор может использовать любые средства связи. Каждый участник подключаются по ссылке, которую получают от инициатора встречи. Чтобы отключиться от комнаты нужно просто закрыть страницу в браузере. Комната будет доступна пока из нее не уйдет последний участник.

Идентификатор комнаты генерируется автоматически, но его можно сменить вручную. Чем сложней идентификатор - тем меньше вероятности того, что кто-то посторонний попадет в комнату. Для дополнительной защиты инициатор может установить пароль на чат комнату и ограничить количество участников, которые смогут к ней подключится. Так же для визуального контроля, количество участников и их роли отображаются в информационной панели для пользователя.

Встречи могут иметь более сложную структуру. Например, если для группы важно к процессу подключить экспертов, которые не будут принимать участия в самом обсуждении, но могут наблюдать за процессом и выдать свое заключение. В этом случае, в DD Talk инициатор встречи активирует режим наблюдения и отправляет экспертам ссылку для подключения. Наблюдатели видят и слышат участников в комнате, но не могут участвовать в общении.Участники в комнате видят и слышат друг друга, но не видят и не слышат наблюдателей. Также наблюдатели не могут общаться между собой. Этот режим инициатор может отключить в любой момент.

Не все пользователи имеют устойчивую интернет связь и технику с мощным и современным процессором. В DD Talk каждый участник имеет возможность самостоятельно вручную регулировать полосу пропускания и разрешение видео потока, чтобы уменьшить качество сигнала настолько, чтобы другой участник со слабыми техническими возможностями мог участвовать в разговоре. В сложных ситуациях когда группа не хочет отвлекаться, будет удобно если за процессом связи будет следить отдельный человек - администратор, который управлять настройками подключения всех участников. В любой момент инициатор может активировать (и деактивировать) режим администратора и передать ему ссылку для подключения. В панели администратора отображаются идентификаторы всех участников. Администратор может всем или каждому отдельно послать команду для отключения, перезагрузки сервиса, изменить параметры подключения, включить/выключить микрофон, отключить участников.

Сервис корректно работает в браузере Google Chrome начиная с 69 версии и выше. Сервис предоставляется бесплатно, без каких-либо гарантий - по принципу "как есть". Если вы обнаружили ошибку в работе сервиса и/или есть предложение по его развитию, пожалуйста, напишите команде разработчиков на адрес ddp.talk [@] gmail.com.

Security

Безопасность обмена данных в пиринговой сети обеспечивается технологиями, которые предотвращают перехват, прослушивание, вмешательство или подделку содержимого сообщения. В работе сервиса не используется cookies. TURN сервер помогает пользователям скрывать свой реальный IP-адрес во время разговора. Сервис DD Talk не собирает и не хранит данных о своих пользователях. About service

DD Talk разрабатывается с начала 2018 года группой волонтеров для организации безопасного канала связи Платформы Донбасский Диалог. Платформа Донбасский Диалог это международная миростроительная инициатива для разрешения вооруженного конфликта в Украине. С 2015 года в очень острой фазе военного конфликта Донбасский Диалог открыл возможность фасилитируемого диалога людям, находящимся по разные стороны линии разделения и не имеющих другой возможности для встречи и беседы с представителями “противной” стороны, для обсуждения закрытых и/или табуированных в этих сообществах тем. DD Talk создает анонимность и гарантирует безопасность для онлайн диалога представителей сообществ разделенных военным конфликтом. Чтобы узнать больше о методике онлайн диалога и платформе Донбасский Диалог перейдите по ссылке https://online-dialogue.org
